Команда man выполняет функцию справочника по командам Linux. Для вызова справки о какой-то команде нужно написать - man <команда>.

Параметр -f в команде man позволяет посмотреть краткое описание справочной страницы.

Параметр -K позволяет выполнять поиск по самим справочным страницам.

Команда info является альтернативой команде man. Но в данном случае при выводе информации можно пользоваться приведенными гиперссылками.

Параметр -f в команде info выдает более краткую информацию по запросу.

параметр - а находит все файлы, соответствующие справочному руководству.

Команда more предназначена для постраничного просмотра файлов. Если содержимое файла слишком велико, чтобы помещаться на одном экране, она отображает содержимое страницы за страницей. Но прокручивать текст можно только вперед.

Также в команду more можно перенаправлять вывод других команд.

Команда less выполняет похожие функции, но позволяет прокручивать текст в обоих направлениях. Также из команда может осуществлять поиск в обоих направлениях, открывать файл в редакторе.

Символические ссылки можно сравнить с ярлыками. Они содержат адрес нужного файла. При открытии ссылки открывается нужный файл или папка. При удалении файла/папки ссылка сохранится, но будет вести в никуда.

Жесткая ссылка это еще одно имя для файла. При создании жесткой ссылки создается дополнительный указатель на существующий файл, но не его копия.


Создание символической ссылки
ln -s labwork2 softlink

Создание жесткой ссылки
ln labwork2 hardlink

Переименование жесткой и символической ссылок

После удаления файла labwork2 символическая ссылка перестала работать, так как была по сути ярлыком. Жесткая ссылка по-прежнему работает, так как она продолжает указывать на нужный участок диска, где хранятся данные.

Символьные (байт-ориентированные) устройства читают и записывают данные в виде потока байтов. Ими бывают последовательные и параллельные порты, накопители на магнитной ленте, терминалы и звуковые платы.

Блочные (блок-ориентированные) устройства читают и записывают данные блоками фиксированного размера. Блочные устройства предоставляют произвольный доступ к своим данным. Пример - жесткий диск.

Тип устройства можно определить с помощью команд

ls /dev/
lsblk
fdisk -l
